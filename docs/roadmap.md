# Roadmap

1. **Milestone 1 (completed):** repository and development environment scaffold.
2. **Milestone 2 (completed):** validated YAML, environment, and CLI
   configuration loading without hardware I/O.
3. **Milestone 3 (completed):** immutable runtime planning, injected component
   contracts, deterministic lifecycle coordination, and health-state foundation
   without adapters or connectivity checks.
4. **Next, separately approved milestone:** establish baseline HyperHDR, WLED,
   and physical-installation validation only
   after a separately approved hardware milestone.
5. Consider multi-zone orchestration only after the baseline path is proven.
